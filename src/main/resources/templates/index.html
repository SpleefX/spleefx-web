SpleefX
The most powerful, compatible, simple yet advanced plugin
to run your spleef, splegg and bow spleef minigames.
  • Customizable
    The plugin comes with a bunch of different configuration files and options to allow you to fully control your gameplay!
    Every game mode gets its own set of options for a fully customizable experience
  • Simple
    Despite the countless features and configurable options, we ensure that the process of setting up and using is as simple as possible, to all users including those with zero background experience.

    We love GUIs.
  • Performant
    SpleefX was built with extensive effort put in optimizing it and ensuring it to be as fast and lightweight as possible. Through use of multi-threading and smart cache, SpleefX comes off as light as feather.
    Who likes lag? Definitely not us.
  • Powerful
    Through the upcoming statistics web UI, in-game music tracks, and a lot of "small" details, we have all you need to take your gaming experience to the next level.

    Feeling like gaming today? We got you.
